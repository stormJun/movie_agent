# 第二层：核心子系统

> **目标读者**：架构师、开发者
> **总阅读时间**：3-5 小时
> **难度等级**：⭐⭐⭐

## 📋 本层内容

深入理解各核心子系统的工作原理、技术实现和使用方法。

### 文档列表

1. [Agent 系统](./Agent系统.md) - BaseAgent 架构、5种 Agent 实现、LangGraph 编排（60分钟）
2. [知识图谱构建](./知识图谱构建.md) - 实体关系提取、图谱构建流程、增量更新（60分钟）
3. [搜索引擎](./搜索引擎.md) - Local/Global/Hybrid/DeepResearch 四种搜索策略（50分钟）
4. [记忆系统](./记忆系统/README.md) - mem0 集成方案、分层落点、端口设计与落地计划（30分钟）
5. [社区检测](./社区检测.md) - Leiden 和 SLLPA 算法、社区摘要生成（40分钟）
6. [评估系统](./评估系统.md) - 20+ 评估指标、测试框架、性能分析（40分钟）

## 🎯 学习目标

完成本层后，您将能够：

- ✅ 理解每个子系统的核心功能和实现原理
- ✅ 掌握各子系统的配置和使用方法
- ✅ 理解子系统之间的协作关系
- ✅ 能够针对特定需求选择合适的组件
