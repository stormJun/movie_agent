# 评估系统

> 本文说明评估模块的位置、作用与最小运行方式。

## 位置

- 评估模块：`backend/graphrag_agent/evaluation/`
- 评估脚本与用例：`test/evaluation/`

## 作用

用于衡量不同 Agent / 检索策略在特定数据集上的效果与性能，包括：

- 结果质量（如相关性、覆盖度等指标）
- 时延、吞吐、稳定性
- 关键链路（缓存、向量检索、图查询）健康度

## 最小运行

项目默认单元测试仍以 `test/` 为主：

```bash
bash scripts/test.sh
```

如需运行评估目录下的脚本，请按 `backend/graphrag_agent/evaluation/readme.md` 的说明选择对应入口。
