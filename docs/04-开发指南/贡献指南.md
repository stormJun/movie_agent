# 贡献指南

## 开发环境

- Python 3.10+
- 建议使用虚拟环境（`conda` 或 `venv`）
- 依赖安装：`pip install -r requirements.txt`

## 代码风格

- 遵循 PEP 8
- 建议使用 `black` / `isort`（按默认风格即可）

## 测试

提交前至少运行：

```bash
bash scripts/test.sh
```

## 文档

- 新增配置项时同步更新 `.env.example` 与 `assets/start.md`
- 文档中的路径尽量使用仓库相对路径（避免 `/Users/...`）
