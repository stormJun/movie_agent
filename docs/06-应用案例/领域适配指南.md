# 领域适配指南

> 目标：把通用 GraphRAG 能力适配到你的领域（实体/关系/查询习惯/数据源）。

## 1. 定义领域 Schema

优先通过 `.env` 配置领域 Schema（对应 `backend/config/rag.py`），无需改代码：

- `GRAPH_THEME`
- `GRAPH_ENTITY_TYPES`（逗号分隔）
- `GRAPH_RELATIONSHIP_TYPES`（逗号分隔）

## 2. 准备领域数据

- 将领域文档放入 `files/`（支持子目录）
- 首次建议用少量文件做 smoke test

## 3. 构建图谱与索引

```bash
bash scripts/py.sh infrastructure.integrations.build.main
```

## 4. 验证与迭代

- 用典型问题做回归（前端或 API）
- 迭代 Schema 与提示词模板

## 5. 应用案例

- 电影推荐知识图谱：`docs/06-应用案例/电影推荐知识图谱设计文档.md`
