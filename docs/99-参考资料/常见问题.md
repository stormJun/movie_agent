# 常见问题（FAQ）

## 1. Neo4j 无法连接

- 先确认已启动：`docker compose up -d`
- 检查 `.env` 中的 `NEO4J_URI/NEO4J_USERNAME/NEO4J_PASSWORD`

## 2. LLM/Embedding 调用失败

- 检查 `.env` 中的 `OPENAI_API_KEY` 与 `OPENAI_BASE_URL`
- 如果使用中转/代理，确认网络可达与鉴权正确

## 3. TMDB 数据获取失败（电影案例）

- 检查 `.env` 中是否配置 `TMDB_API_TOKEN` 或 `TMDB_API_KEY`
- 网络受限时配置 `HTTP_PROXY/HTTPS_PROXY`
- 预处理脚本：`python3 data/movie/movie_data_preprocessing.py --help`
